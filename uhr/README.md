# uhr

Copia de la suite de medicion de tiempos 
(https://github.com/leonardlover/uhr, Autor: Leonardo Lovera, MIT
License, ver [LICENSE](LICENSE)).

`utils.cpp` incluye `quartiles()`, la funcion que usa
[harness/arnesExperimentos.cpp](../harness/arnesExperimentos.cpp) para
calcular los cuartiles de cada conjunto de mediciones. `uhr.cpp` es el
programa original de la suite (un cronometro configurable de proposito
general); se deja aqui como referencia, sin modificar.
